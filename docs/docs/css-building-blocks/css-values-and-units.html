            numeric:
                <integer>
                    whole
                <number>
                    decimal, it may or may not have a decimal point with a fractional component
                <dimension>
                    维度, a <number> with a unit; 
                    umbrella category: length, angle, time, resolution types
                <percentage>
                    a fraction of some other value, %, always relative to another quantity
        
            lengths

                absolute length units
                !as reference, not necessarily be remembered
                    cm centimeters 1cm = 96px/2.54
                    mm millimeters 1mm = 1/10th of 1cm
                    Q quarter-millimeters 1Q = 1/40th of 1cm
                    in inches 1in = 2.54cm = 96px
                    pc picas 1pc = 1/6th of 1in
                    pt points 1pt = 1/72th of 1in
                    px pixels 1px = 1/96th of 1in

            hsl and hsla values

            slightly less well-supported than rgb  (not supported on old versions of IE)
            designers like it
            hsl() - hsl(hue, saturation, lightness)

            hue: the base shade of the color. 
            This takes a value between 0 and 360, representing the angles round a color wheel.
            shade 色度 example:也许讨论颜色时使用色彩（tint）、色度（shade）和色调（tone）这三个术语是有好处的。 把色调、饱和度和亮度结合起来，我们就可以创建任何一种人眼哦可见的颜色。
            色相（hue）：指饱和度和亮度都为100%的颜色。我们肉眼所见到于是能说出来的颜色就叫色相

            saturation: how saturated is the color? 0-100%, 0 no color (appear completely black), 100% (full color saturation) 
            饱和度（Saturation）：是指色调深浅的程度。各种单色光饱和度最高，单色光中掺入的白光愈多，饱和度愈低，白光占绝大部分时，饱和度接近于零

            lightness: how light or bright is the color? 0-100%, 0 no light (appear completely black), 100% (full light, completely white)
            视亮度（Lightness）:是指从白色外表到黑色外表的觉得延续体。它由物体外表的反射系数决议的，而与物体的照度有关。

            hsla: hsl + alpha channel
