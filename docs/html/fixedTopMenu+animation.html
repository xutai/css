How TO - Slide Down a Bar on Scroll 


        transition: top 0.3s;

    window.onscroll = function() {scrollFunction()};

    function scrollFunction() {
    if (document.body.scrollTop > 20 || document.documentElement.scrollTop > 20) {
    document.getElementById("navbar").style.top = "0";
    } else {
    document.getElementById("navbar").style.top = "-50px";
    }
    }

How TO - Sticky/Affix Navbar

Element offsetTop Property if body { margin: 10px;} and the offsetParent is body if relative position is 100px then offsetTop is 110px

Window pageXOffset pageYOffset Properties

use js to add class or remove class, and css selector .sticky + .content 


        window.onscroll = function() {myFunction()};

    var navbar = document.getElementById("navbar");
    var sticky = navbar.offsetTop;

    function myFunction() {
    if (window.pageYOffset >= sticky) {
    navbar.classList.add("sticky")
    } else {
    navbar.classList.remove("sticky");
    }
    }
